# Critter Mound

Critter Mound is an incremental browser game featuring critter breeding, worker management, combat, and prestige mechanics. Originally created in 2014, it is being modernized to current web standards.

## Features

- **Breeding System** — Trait inheritance, mutations, and genetic scoring
- **Resource Management** — Dirt, grass, and sod production chains
- **Worker System** — Miners, farmers, carriers, and factories
- **Combat System** — Army management, battles, and level progression
- **Prestige Mechanics** — Heir system with princesses and princes
- **Dark Mode** — Toggle between light and dark themes

## Quick Start

```bash
npm install
npm run dev
```

The dev server opens at `http://localhost:3000` with hot module replacement.
